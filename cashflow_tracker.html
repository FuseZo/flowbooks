📤 ส่งออกข้อมูล

ส่งออกรายการธุรกรรมทั้งหมดในรูปแบบ CSV หรือ Excel เพื่อใช้กับ Google Sheets, Excel หรือโปรแกรมอื่นๆ

📥 นำเข้าข้อมูล

นำเข้าข้อมูลจากไฟล์ CSV ที่ export ไว้ก่อนหน้า (ต้องมีคอลัมน์ตามรูปแบบที่กำหนด)

💾 สำรองข้อมูล

บันทึกข้อมูลทั้งหมด (ธุรกรรม + งบประมาณ + ตั้งค่า) เป็นไฟล์ JSON เพื่อกู้คืนในภายหลัง

🐙 GitHub Gist Database

เชื่อมต่อ GitHub Gist เพื่อ sync ข้อมูลข้ามอุปกรณ์ได้ — ใช้งานได้ฟรี ไม่ต้องมี server

🗑️ ล้างข้อมูล

ลบข้อมูลทั้งหมดออกจากระบบ ไม่สามารถกู้คืนได้หากไม่ได้สำรองข้อมูลไว้
